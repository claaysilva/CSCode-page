# CSCode - Landing Page

Landing page institucional da CSCode focada em automacao, integracoes e implementacao de sistemas orientados a receita.

## Tecnologias

- HTML5
- CSS3
- JavaScript (vanilla)

## Estrutura de Arquivos

- `index.html`: pagina principal
- `style.css`: estilos globais e responsivos
- `script.js`: interacoes de navegacao/menu
- `termos.html`: pagina de termos de uso
- `privacidade.html`: pagina de politica de privacidade
- `logo.png`: logo da marca

## Funcionalidades

- Sessao de servicos com oferta principal e prova social
- Navegacao responsiva com menu mobile
- CTA principal para WhatsApp abrindo em nova aba
- Botao flutuante de suporte via WhatsApp
- Rodape institucional com dados da empresa e links legais

## Como Executar Localmente

1. Abra a pasta do projeto no VS Code.
2. Abra o arquivo `index.html` no navegador.

Opcional (com Live Server):

1. Instale a extensao Live Server no VS Code.
2. Clique com o botao direito em `index.html`.
3. Selecione `Open with Live Server`.
